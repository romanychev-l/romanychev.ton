1. Стандартный пункт. См предыдущий день. Добавлю подпункт позавтракать сюда.
2. Посетить все пары(вроде поправился).
3. Переписать конспекты пропущенных лекций по матану.
4. Билеты по матану написать с 30 по 40й.
5. Заказать справки в учебном отделе.
6. Обновить студенческий в 10 общаге.
7. Разобраться с учебники. Сдать книгу по Алгоритмам. Сразу забрать ее. Взять нужную для учебы литературу.
